This list shows the names associated with magically enhanced items. See the Item Format page for the location of the ID's.

Notice: This is just a preliminary list; it doesn't have a fraction of the possible names. Also, some names are associated with more than one ID. This has to do with the level of the enhancement, and to what items the prefix or suffix can be applied.
